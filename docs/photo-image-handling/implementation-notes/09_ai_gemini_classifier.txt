========================================
AI GEMINI CLASSIFIER (TIER 1)
========================================
File: src/ai/gemini_classifier.py
Purpose: Fast item classification using Google Gemini

Key Method: analyze_item(photos: List[Photo]) -> Dict[str, Any]

Returns:
{
    "item_name": str,
    "brand": str,
    "franchise": str,  # e.g., "Star Wars", "MLB", "Pokemon"
    "category": str,  # electronics, toys, apparel, home_goods, etc.
    "description": str,  # simple description
    "collectible": bool,  # YES/NO - triggers deep analysis
    "collectible_confidence": float,  # 0.0 to 1.0
    "collectible_indicators": List[str],  # what triggered collectible flag
    "estimated_value_low": float,
    "estimated_value_high": float,
    "detected_keywords": List[str],
    "sku_upc": str,  # if visible
    "logos_marks": List[str],
    "condition": str,  # new, like_new, good, fair, poor
    "color": str,
    "size": str,
    "material": str,
    "suggested_title": str,
    "suggested_price": float,
    "market_analysis": {
        "sell_through_rate": float,
        "demand_level": str,  # High, Medium, Low
        "days_to_sell_estimate": int,
        "average_sold_price": float,
        "market_insights": List[str]
    }
}

Key Features:
- Fast classification (Gemini is quick and cheap)
- Collectible detection (triggers Tier 2 enhanced scan)
- Market analysis integration
- Form auto-fill data extraction

Implementation:
- Uses Google Gemini Vision API
- Limits to 4 photos for speed
- Base64 encodes images
- Returns JSON with all detected attributes
