========================================
AI CLAUDE ANALYSIS (TIER 2)
========================================
File: src/collectibles/recognizer.py, src/enhancer/ai_enhancer.py
Purpose: Deep collectible analysis using Claude

Key Methods:
1. analyze_for_collectibles_claude() - Collectible identification
2. deep_analyze_collectible() - Detailed collectible analysis
3. analyze_photos_claude() - General photo analysis

Returns for Collectibles:
{
    "is_collectible": bool,
    "category": str,  # toys, cards, coins, stamps, etc.
    "item_name": str,
    "brand": str,
    "franchise": str,
    "year": int,
    "estimated_value_low": float,
    "estimated_value_high": float,
    "authentication": {
        "is_authentic": bool,
        "confidence": float,
        "authentication_markers": List[str],
        "red_flags": List[str]
    },
    "rarity": {
        "rarity_level": str,
        "print_run": str,
        "variants": List[str]
    },
    "market_analysis": {
        "current_market_value_low": float,
        "current_market_value_high": float,
        "market_trend": str,
        "demand_level": str
    }
}

Returns for Cards:
{
    "card_name": str,
    "player_name": str,  # for sports cards
    "set_name": str,
    "card_number": str,
    "rarity": str,
    "year": int,
    "brand": str,
    "is_rookie_card": bool,
    "is_graded": bool,
    "grading_company": str,
    "grading_score": float,
    "estimated_value_low": float,
    "estimated_value_high": float
}

Key Features:
- Deep analysis (more accurate than Gemini)
- Authentication checking
- Rarity analysis
- Market pricing integration
- Database storage integration

Implementation:
- Uses Anthropic Claude Vision API
- Converts images to JPEG/PNG/GIF before sending
- Limits to 4 photos
- Base64 encodes images
- Returns comprehensive JSON analysis
