Quick Start Guide

Get organized in 5 minutes:
  1. Choose your storage type (Clothing or Cards)
  2. Create bins (A, B, C, etc.)
  3. Add sections to each bin (1, 2, 3, etc.)
  4. Add items and get Storage IDs automatically
  5. Use Storage Map to find items instantly!
Pro Tip: Start with ONE bin to test the system before creating your full setup.

Card Category Codes

When adding cards, use these category codes for your Storage IDs:

Sports Cards:
  • FB - Football (NFL)
  • BB - Basketball (NBA)
  • BSB - Baseball (MLB)
  • NHL - Hockey
  • SOC - Soccer
Trading Card Games:
  • PKMN - Pokémon
  • MTG - Magic: The Gathering
  • YGO - Yu-Gi-Oh!
Entertainment:
  • WWE - Wrestling
  • MRV - Marvel
  • SW - Star Wars
Other:
  • OTH - Other/Misc
Organization Strategies
By Rarity/Value

Organize high-value cards separately for security and quick access.

Example:
Bin A: $100+ cards
Bin B: $20-$99 cards
Bin C: Bulk cards
By Franchise/Sport

Keep each game or sport in separate bins.

Example:
Bin A: Pokémon
Bin B: Magic: The Gathering
Bin C: Sports cards
By Set/Year

Organize by release set or year for completionists.

Example:
Bin A: 2024 releases
Bin B: 2023 releases
Bin C: Vintage
Recommended: Start with "By Franchise/Sport" for most flexibility. You can always reorganize later!
Physical Storage Tips
For Trading Cards:
  • Binders: Best for high-value cards ($5+), easy to browse
  • Storage Boxes: Best for bulk, sorted inventory
    • 100-count: Small sets, quick access
    • 660-count: Medium collections
    • 3200-count: Bulk storage
  • Top Loaders + Bins: Maximum protection for $10+ cards
  • Deck Boxes: Complete decks, playsets
For Clothing:
  • Plastic Bins: Stackable, labeled, transparent lids
  • Hanging Racks: Quick access for frequently listed items
  • Shelving Units: Organize by size or brand
  • Vacuum Bags: Seasonal items, long-term storage
Labeling Best Practices
Label Everything!
  1. Bins: Large labels on front and top (A, B, C, etc.)
  2. Sections/Compartments: Number or letter each section (1, 2, 3, etc.)
  3. Individual Items: Use Storage IDs from ResellGenie
Cards Example:
PKMN-A1-045 means:
  • PKMN = Pokémon card
  • A1 = Bin A, Section 1
  • 045 = Card #45 in that section
Clothing Example:
BIN-A-COMP-2-007 means:
  • BIN-A = Bin A
  • COMP-2 = Compartment 2
  • 007 = Item #7
Pro Tip: Use a label maker for professional-looking labels. Write Storage IDs on inventory tags attached to items!
Complete Workflow Example
Scan → Organize → Sell → Ship
  1. Scan Item: Use "Scan & Add" to take photos and analyze with AI
  2. Add to Vault: Item saved with details and value estimate
  3. Assign Storage: Place item in physical bin, assign Storage ID
  4. Create Listing: When ready to sell, click "Create Listing"
  5. Item Sells: Storage ID shown in notification
  6. Find & Ship: Use Storage Map to locate item instantly!
Real Example:

Day 1: Scan Charizard Pokémon card → AI detects it's worth $120 → Save to vault → Put in Bin A, Section 1 → Assign PKMN-A1-012

Day 5: Ready to sell → Open vault → Click "Create Listing" → Post to eBay

Day 7: Card sells! → Notification shows PKMN-A1-012 → Use Storage Map to find → Go to Bin A, Section 1, Card #12 → Ship within minutes!
